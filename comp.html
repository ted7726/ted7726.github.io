About

This website
I created this website because I realized every time I write regular expression command, I am not sure what the code exactly matches, whether it is a word document or hundreds of lines of code.

Existing tools weren't purpose-built for instant match with side by side to compare the changes. I created regular expression online tool both to save myself time, and to save the time of others who want to test the regular expression.
